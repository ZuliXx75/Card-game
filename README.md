# Card-game

## Overview

Guess the Card Game is an interactive card guessing game where players select a card from a dropdown menu and then choose one of the cards displayed on the screen. The deck shuffles randomly each round, making every game different. When a card is clicked, it flips over to reveal whether the player guessed correctly or not.

## Features

* Interactive card gameplay
* Randomized card shuffling
* Card flip animations
* Score feedback
* Dynamic user interface
